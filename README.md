# WiFi Motion Detection System

A real-time Wi-Fi sensing system developed with Arduino UNO R4 WiFi boards.

The project monitors Wi-Fi RSSI values from multiple Arduino boards connected to the same access point. Signal variations caused by human movement are transmitted to a computer over UDP and visualized through a Python-based graphical interface.

## Features

- Real-time RSSI monitoring
- Multi-board support (UNO1, UNO2, UNO3)
- UDP communication
- Human motion detection
- Live signal visualization
- Cross-platform Python GUI

## Hardware

- Arduino UNO R4 WiFi ×3
- Wi-Fi Access Point
- Windows Laptop

## Software

- Python
- Tkinter
- Matplotlib
- NumPy
- Linux (Arduino UNO R4 WiFi)
